# Matematicas-Discretas-Permutaci-n-y-Combinaci-n-con-Python.
Un programa en python que permite realizar factoriales, permutaciones, combinaciones y el triángulo de pascal sin librerías adicionales. Proyecto hecho para Matemáticas Discretas 2026-1 grupo 3, Universidad Nacional de Colombia.

Para usar el programa, siga las siguientes instrucciones:
1. Descargue el cuaderno ipynb que se encuentra en las release del proyecto, o abra el siguiente link de google Colab:
https://colab.research.google.com/drive/16f2AJdnXfq5uMCJfS9SW2cv9K6vWVH9E?usp=sharing
2. Una vez abierto el programa, encontrará el código y una explicación del  mismo. Ejecute todas las celdas del código para que el programa se inicialice y funcione adecuadamente.
En las celdas dónde se le pidan valores, puede colocar cualquier valor.
3. Cuando tenga inicializadas las celdas de código, en la última sección encontrará el código propiamente, simplemente ejecute la celda con lo que quiera realizar y coloque los valores que quiera para realizar las respectivas operaciones. 

Tenga en cuenta que está descargando el código fuente también, de preferencia no lo modifique si no desea que el programa gente errores. También puede modificar el colab, pero tenga en cuenta que sus cambios se guardarán de forma local. Es posible hacer comentarios en el cuaderno de Colab.

Puede encontrar ejemplos de uso y funcionamiento en la carpeta "funcionamiento" del repositorio.
